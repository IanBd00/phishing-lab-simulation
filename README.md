# phishing-lab-simulation

🔐 Simulação de Phishing em Laboratório Controlado

Projeto desenvolvido como parte da plataforma DIO (Digital Innovation One) com finalidade exclusivamente educacional.

📌 Sobre o Projeto

Este repositório documenta um estudo prático sobre engenharia social e ataques de phishing, realizado em ambiente controlado para fins de aprendizado em Segurança da Informação.

O objetivo foi compreender:

Como ataques de phishing funcionam tecnicamente;<br>
Quais ferramentas são utilizadas em simulações;<br>
Como ocorre a captura de credenciais.

⚠️ Aviso:
Nenhum sistema real, usuário real ou serviço oficial foi utilizado neste projeto. Toda a prática ocorreu em laboratório isolado.

🛠️ Ferramentas Utilizadas:

Kali Linux rodando em uma VM;<br>
Script Setoolkit

🎯 Etapas do Processo:

Acesso root: *sudo su*<br>
Iniciando o setoolkit: *setoolkit*<br>
Tipo de ataque: *Social-Engineering Attacks*<br>
Vetor de ataque: *Web Site Attack Vectors*<br>
Método de ataque: *Credential Harvester Attack Method*<br> 
Método de ataque: *Site Cloner*<br>
